Testing Mermaid Diagrams

Jan 18, 2025
Table of Contents

  • Method 2: Using the Mermaid Shortcode
  • More Complex Example
  • Class Diagram Example

This post demonstrates how to use Mermaid diagrams in your Hugo site. There are two ways to include Mermaid diagrams:

Method 1: Using Code Fences

You can use standard markdown code fences with the mermaid language identifier:

graph TD;
    A-->B;
    A-->C;
    B-->D;
    C-->D;

Method 2: Using the Mermaid Shortcode

You can also use the custom shortcode:

sequenceDiagram participant Alice participant Bob Alice->>John: Hello John, how are you? loop Healthcheck John->>John: Fight against hypochondria end Note right of John: Rational thoughts prevail! John-->>Alice: Great! John->>Bob: How about you? Bob-->>John: Jolly good!

More Complex Example

Here’s a more complex flowchart:

flowchart LR
    A[Start] --> B{Is it?}
    B -->|Yes| C[OK]
    C --> D[Rethink]
    D --> B
    B ---->|No| E[End]

Class Diagram Example

classDiagram
    class Animal {
        +String name
        +int age
        +makeSound()
    }
    class Dog {
        +String breed
        +bark()
    }
    class Cat {
        +boolean indoor
        +meow()
    }
    Animal <|-- Dog
    Animal <|-- Cat

All diagrams should render properly and adapt to your site’s theme automatically.
